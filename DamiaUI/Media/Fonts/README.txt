DamiaUI Media - Fonts Directory

This directory contains font files for the DamiaUI addon.

Font Categories:
- UI fonts for interface elements
- Number fonts for damage/healing displays
- Chat fonts for enhanced readability
- Special fonts for titles and headers

File Formats:
- .ttf (TrueType Font - Recommended)
- .otf (OpenType Font - Supported)

Font Requirements:
- Must include full character sets for supported locales
- Should be legible at small sizes (8-12pt)
- Consider licensing restrictions for distribution

Naming Convention:
- Use descriptive names: DamiaUI_Bold.ttf
- Include weight/style in name: DamiaUI_Light.ttf
- Avoid spaces in filenames

Production Note:
In production, this directory would contain actual font files
such as:
- DamiaUI_Regular.ttf
- DamiaUI_Bold.ttf
- DamiaUI_Numbers.ttf
- etc.

Default WoW fonts can be referenced as:
- "Fonts\\FRIZQT__.TTF" (UI font)
- "Fonts\\ARIALN.TTF" (Arial Narrow)
- "Fonts\\skurri.TTF" (Damage font)
