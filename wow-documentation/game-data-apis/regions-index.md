# Regions Index

## Endpoint
```
GET /data/wow/region/index?namespace=dynamic-us&locale=en_US
```

## Response
```json
{
    "_links": {
        "self": {
            "href": "https://us.api.blizzard.com/data/wow/region/?namespace=dynamic-us"
        }
    },
    "regions": [
        {
            "href": "https://us.api.blizzard.com/data/wow/region/1?namespace=dynamic-us"
        }
    ]
}
```
